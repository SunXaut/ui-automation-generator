# Vue Test Project

这是一个用于测试 vue-test-generator skill 的 Vue 3 项目。

## 技术栈

- Vue 3.4+
- TypeScript
- Vite 5
- Vue Router 4
- Pinia 2

## 组件列表

- `Login.vue` - 登录表单（包含多种事件）
- `Dashboard.vue` - 仪表盘
- `Search.vue` - 搜索页面
- `Home.vue` - 首页
- `About.vue` - 关于页面

## 安装

```bash
npm install
```

## 运行

```bash
npm run dev
```
