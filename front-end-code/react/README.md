# React Test Project

这是一个用于测试 vue-test-generator skill 的 React 项目。

## 技术栈

- React 18+
- TypeScript
- Vite 5
- React Router 6
- Pinia 2

## 组件列表

- `Login.tsx` / `LoginForm.tsx` - 登录表单（包含多种事件）
- `Dashboard.tsx` - 仪表盘
- `Search.tsx` - 搜索页面
- `Home.tsx` - 首页
- `About.tsx` - 关于页面

## 安装

```bash
npm install
```

## 运行

```bash
npm run dev
```
